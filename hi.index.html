1. HTML

HTML (HyperText Markup Language) is used to structure web content.

<!DOCTYPE html>
<html>
  <head>
    <title>My Web Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

2. CSS

CSS (Cascading Style Sheets) is used for styling HTML elements.

body {
  background-color: #121212;
  color: white;
  font-family: Arial, sans-serif;
}
p {
  color: yellow;
  font-size: 18px;
}

3. JavaScript

JavaScript is used to make web pages interactive.

<script>
  function greet() {
    alert("Hello from JavaScript!");
  }
</script>

<button onclick="greet()">Click Me</button>
